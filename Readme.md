# 🎨 Color Palette Studio

A modern Color Palette Generator built with HTML, CSS, and JavaScript.  
Generate beautiful random color palettes, copy HEX codes, and export CSS variables instantly.

---

## ✨ Features

- 🎨 Random color palette generator
- 📋 Click to copy HEX colors
- 🧠 5-color professional palette layout
- 📥 Export palette as CSS variables
- ⚡ Instant generation
- 💻 Clean UI design
- 🚀 Lightweight and fast tool

---

## 📂 Project Structure

```
color-palette-studio/

```

---

## 🚀 How to Use

1. Open `index.html` in browser
2. Click **Generate Palette**
3. Click any color to copy HEX code
4. Click **Export CSS Variables** to download
